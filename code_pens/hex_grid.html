Hexagonal Grids

30 Nov 2018 - Brent Jacobs - See the Pen Hexagonal Grids

See the Pen Hexagonal Grids by Brent Jacobs (@br3nt) on CodePen.

This pen is my attempt at learning how to build hex grids from the Hexagonal Grids tutorial by Red Blob Games.

The example is using the p5.js library.

The hex grid can be oriented such that the hex points or edges face upwards. This is currently done in code.

The grid tracks mouse movements so that the hex currently under the mouse is highlighted. Hexes are also selectable.
